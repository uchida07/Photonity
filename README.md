## サイト概要
### サイトテーマ
<写真が好きな人同士で批評を行うコミュニティサイト>

<カメラが趣味で撮影を行なったりしていますが、同じ趣味でつながり・技術向上を相談できる場所が少ないと感じたため、このテーマにしました。>

### ターゲットユーザ
<カメラ、写真が好きな人>

### 主な利用シーン
<自分の撮った写真を投稿し評価や批評をして同じ利用者とコミュニケーションがとれる>
